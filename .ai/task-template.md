# Task Contract Template

Use this template before starting a non-trivial implementation task.

```text
Task Name:

Business Goal:

User/Operator Outcome:

In Scope:

Out Of Scope:

Files Or Directories Expected To Change:

Files Or Directories That Must Not Change:

Dependent Schemas:

Dependent Pages Or Surfaces:

Dependent Prompts / Tools / Evaluators:

Dependent Data Sources Or Repositories:

Acceptance Criteria:

Testing Requirements:

Eval / Fixture Requirements:

Risk Notes:

Rollback Plan:

Open Questions:
```

Recommended workflow:

1. Fill this contract.
2. Read `.ai/repo-map.md` and the relevant rule/checklist files.
3. Create or update `PLANS.md` if the task is large.
4. Implement one verifiable slice at a time.
